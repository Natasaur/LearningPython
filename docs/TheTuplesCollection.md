#Tuples
These are like lists.  
Tuples are ampther kind of sequence that functions much like a list. They have elements which are indexed starting at 0.
```
x = ('Glenn','Sally','Joseph')
print(x[2])
```
This prints the name "Joseph".

```
y = (1,9,2)
for iter in y:
    print(iter)
```
This prints:
1
9
2
